# typeface-khmer
[![npm][npm]][npm-url]

This is npm package for khmer typefaces.

### Install

```bash
npm install khmer-typeface
yarn add khmer-typeface
```

### Use

require library in your JS code

```javascript
import 'khmer-typeface/index.css';
```

```css
body {
  font-family: 'Khmer-CN-Poipet';
}
```
